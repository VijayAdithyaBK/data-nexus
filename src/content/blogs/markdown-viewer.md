## Introduction

Markdown is the language of the web, but raw text isn't always easy to parse. **Markdown Viewer** renders your `.md` files into beautiful HTML instantly.

---

## Features

-   **Live Preview**: See changes as you type.
-   **Github Flavored**: Supports standard GFM syntax.
-   **Responsive**: Looks great on mobile and desktop.

---

## Tech Stack

-   **Frontend**: React + TypeScript
-   **Build**: Vite
-   **Design**: Tailwind CSS

---

## Conclusion

Essential for writers and developers who want to check their documentation before pushing to production.
