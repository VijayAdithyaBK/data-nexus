## Introduction

Why install heavy IDEs for quick HTML/CSS experiments?

**Online Code Editor** is a web-based tool that lets you write, edit, and preview code instantly in your browser.

---

## The Features

-   **Live Preview**: See changes instantly as you type.
-   **Syntax Highlighting**: Read code comfortably.
-   **Lightweight**: No installation required, runs entirely on the client side.

---

## Tech Stack

-   **React**: For the interactive interface.
-   **TailwindCSS**: For a sleek, modern look.
-   **Vite**: For lightning-fast development and build.

---

## Conclusion

This project serves as a perfect playground for developers to test snippets or for beginners to learn web development without the hassle of setting up a local environment.
