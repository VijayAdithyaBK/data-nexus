## Introduction

Developers deal with colors every day. Hex, RGB, HSL—converting between them shouldn't require Google.

**Colorly** is a lightweight JavaScript package for handling color operations programmatically.

---

## Features

-   **Tiny Footprint**: Minimal dependencies.
-   **Type Definitions**: Includes `index.d.ts` for TypeScript support.
-   **Ease of Use**: Simple API for color conversion.

---

## Tech Stack

-   **Runtime**: Node.js
-   **Language**: JavaScript / TypeScript

---

## Conclusion

Sometimes the best tools are the small, focused ones. Colorly solves the specific problem of color manipulation without adding bloat to your project.
