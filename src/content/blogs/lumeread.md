## Introduction

Reading online is often plagued by ads, popups, and clutter. **LumeRead** strips all that away, providing a serene environment for your text.

---

## Features

-   **Distraction-Free**: Clean layout focused solely on typography.
-   **Copy-Paste-Read**: Simple workflow to process any text.
-   **Modern UI**: Built with Shadcn UI for accessible and beautiful components.

---

## Tech Stack

-   **Framework**: React (Vite)
-   **Language**: TypeScript
-   **Styling**: Tailwind CSS
-   **Components**: Shadcn UI

---

## Conclusion

LumeRead turns reading from a chore into a focused experience.
