## Introduction

While static sites are popular for portfolios, sometimes you need backend power. **RP Portfolio** uses **Django** to manage projects and blog posts dynamically via an admin panel.

---

## Features

-   **Dynamic Content**: Add projects without redeploying code.
-   **Admin Panel**: Built-in Django admin for content management.
-   **Database**: SQLite/PostgreSQL for data persistence.

---

## Tech Stack

-   **Framework**: Django (Python)
-   **Templating**: Django Templates (Jinja2-like)

---

## Conclusion

Demonstrates how to build a content-managed website using Python's battery-included framework.
